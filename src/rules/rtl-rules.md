# RTL Rules

## بایدها

- ریشه سایت باید `dir="rtl"` داشته باشد.
- spacingهای جهت‌دار با `padding-inline`, `margin-inline`, `inset-inline` نوشته شوند.
- متن فارسی باید `text-align: start` داشته باشد.
- اعداد، کدها، URLها و ایمیل‌ها می‌توانند `dir="ltr"` داشته باشند.

## نبایدها

- استفاده غیرضروری از `left` و `right`
- mirror نکردن آیکن‌های جهت‌دار
- ساخت layout اول LTR و سپس اصلاح سطحی با RTL
