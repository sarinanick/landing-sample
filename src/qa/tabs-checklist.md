# Tabs QA Checklist

- [ ] Primary tabs دارد.
- [ ] Secondary tabs دارد.
- [ ] active state دارد.
- [ ] inactive state دارد.
- [ ] disabled state دارد.
- [ ] hover state دارد.
- [ ] focus-visible state دارد.
- [ ] pressed state دارد.
- [ ] indicator فعال درست نمایش داده می‌شود.
- [ ] Tab و TabPanel با aria درست وصل شده‌اند.
- [ ] در RTL درست نمایش داده می‌شود.
- [ ] overflow افقی در تعداد زیاد تب‌ها قابل مدیریت است.
