# Tooltip QA Checklist

- [ ] plain tooltip دارد.
- [ ] rich tooltip دارد.
- [ ] top placement دارد.
- [ ] bottom placement دارد.
- [ ] start placement دارد.
- [ ] end placement دارد.
- [ ] hover trigger کار می‌کند.
- [ ] focus trigger کار می‌کند.
- [ ] Escape tooltip را می‌بندد.
- [ ] role="tooltip" دارد.
- [ ] aria-describedby درست تنظیم می‌شود.
- [ ] در RTL درست نمایش داده می‌شود.
- [ ] rich tooltip عنوان و action اختیاری دارد.
