# SearchBar QA Checklist

- [ ] collapsed/default state دارد.
- [ ] expanded/active state دارد.
- [ ] disabled state دارد.
- [ ] loading state دارد.
- [ ] clear button دارد.
- [ ] suggestions نمایش داده می‌شوند.
- [ ] recent searches نمایش داده می‌شوند.
- [ ] results نمایش داده می‌شوند.
- [ ] empty state دارد.
- [ ] Enter search را اجرا می‌کند.
- [ ] Escape پنل را می‌بندد.
- [ ] aria-controls و aria-expanded تنظیم می‌شوند.
- [ ] native input حفظ شده است.
- [ ] در RTL درست نمایش داده می‌شود.
