# Navigation Bar QA Checklist

- [ ] 3 تا 5 آیتم را درست نمایش می‌دهد.
- [ ] active state دارد.
- [ ] inactive state دارد.
- [ ] disabled state دارد.
- [ ] hover state دارد.
- [ ] focus-visible state دارد.
- [ ] pressed state دارد.
- [ ] icon و label هر آیتم نمایش داده می‌شود.
- [ ] badge درست نمایش داده می‌شود.
- [ ] aria-current برای آیتم فعال تنظیم می‌شود.
- [ ] در RTL درست نمایش داده می‌شود.
- [ ] fixed bottom helper کار می‌کند.
