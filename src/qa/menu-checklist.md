# Menu QA Checklist

- [ ] menu با trigger باز و بسته می‌شود.
- [ ] بیرون کلیک کردن menu را می‌بندد.
- [ ] Escape menu را می‌بندد.
- [ ] ArrowDown و ArrowUp فوکوس را جابه‌جا می‌کند.
- [ ] role="menu" دارد.
- [ ] role="menuitem" برای آیتم‌ها دارد.
- [ ] leading icon پشتیبانی می‌شود.
- [ ] trailing text پشتیبانی می‌شود.
- [ ] disabled item درست کار می‌کند.
- [ ] destructive item درست نمایش داده می‌شود.
- [ ] divider درست نمایش داده می‌شود.
- [ ] bottom-start و bottom-end کار می‌کنند.
- [ ] در RTL درست نمایش داده می‌شود.
