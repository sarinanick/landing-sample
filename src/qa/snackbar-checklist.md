# Snackbar QA Checklist

- [ ] open/closed state درست کار می‌کند.
- [ ] auto-dismiss با duration کار می‌کند.
- [ ] close button کار می‌کند.
- [ ] action button کار می‌کند.
- [ ] role status برای پیام معمولی دارد.
- [ ] role alert برای error دارد.
- [ ] bottom position دارد.
- [ ] top position دارد.
- [ ] neutral tone دارد.
- [ ] success tone دارد.
- [ ] error tone دارد.
- [ ] warning tone دارد.
- [ ] multiline mode دارد.
- [ ] hover/focus/pressed برای actionها وجود دارد.
- [ ] در RTL درست نمایش داده می‌شود.
