# Dialog QA Checklist

- [ ] حالت open/closed درست کار می‌کند.
- [ ] scrim در modal dialog نمایش داده می‌شود.
- [ ] کلیک روی scrim طبق تنظیمات می‌بندد.
- [ ] Escape dialog را می‌بندد.
- [ ] role="dialog" دارد.
- [ ] aria-labelledby برای title تنظیم می‌شود.
- [ ] aria-describedby برای supporting text تنظیم می‌شود.
- [ ] actionها hover state دارند.
- [ ] actionها focus-visible state دارند.
- [ ] actionها disabled state دارند.
- [ ] small / medium / large size پشتیبانی می‌شود.
- [ ] در RTL درست نمایش داده می‌شود.
- [ ] focus اولیه به دکمه مناسب منتقل می‌شود.
