# Page QA Checklist

برای هر صفحه بررسی شود:

- [ ] صفحه `dir="rtl"` دارد.
- [ ] همه رنگ‌ها از tokenها آمده‌اند.
- [ ] هیچ Button دستی وجود ندارد.
- [ ] هیچ Text Input دستی وجود ندارد.
- [ ] فونت فارسی درست است.
- [ ] موبایل و دسکتاپ بررسی شده‌اند.
- [ ] focus state قابل مشاهده است.
- [ ] contrast قابل قبول است.
- [ ] dark mode خراب نمی‌شود.
