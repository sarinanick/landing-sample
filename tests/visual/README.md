# Visual Testing Scaffold

این فولدر برای تست‌های تصویری آماده شده است.

## هدف

```txt
Storybook Story
    ↓
Playwright Screenshot
    ↓
Figma Baseline
    ↓
Visual Diff
```

## کارهای بعدی

- نصب Playwright
- اجرای Storybook در CI
- گرفتن screenshot از storyها
- ذخیره baseline از Figma
- مقایسه diff
