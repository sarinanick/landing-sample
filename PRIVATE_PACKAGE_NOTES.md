# Private Package Notes

## Package name

```txt
@ali/persian-material-system
```

## وضعیت

فعلاً package به صورت private تنظیم شده است:

```json
{
  "private": true
}
```

## دلیل

پروژه برای استفاده شخصی است و قرار نیست public منتشر شود.

## اگر بعداً خواستی private package بسازی

گزینه‌ها:

- GitHub Packages
- GitLab Package Registry
- npm private package
- استفاده مستقیم از GitHub private repo

## نکته

هیچ secret یا token نباید داخل پروژه ذخیره شود.
