# Build System

## هدف

این فولدر مستندات build داخلی پروژه را نگه می‌دارد.

## خروجی هدف

```txt
dist/
  index.js
  style.css
  types/
```

## دستورات

```bash
npm run build
npm run build:types
npm run build:lib
```

## استفاده خصوصی

این build برای private package یا استفاده مستقیم در پروژه‌های شخصی طراحی شده است.
