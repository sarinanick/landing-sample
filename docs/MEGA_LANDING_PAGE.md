# Mega Landing Page

## هدف

این صفحه برای تست ترکیب کامپوننت‌های Design System ساخته شد.

## مسیر

```txt
src/examples/mega-landing-page/
  MegaLandingPage.tsx
  MegaLandingPage.css
  index.ts
```

- TopAppBar
- SearchBar
- Button
- IconButton
- Tooltip
- Badge
- Menu
- Card
- Carousel
- Chip
- SegmentedButton
- Slider
- Checkbox
- Switch
- TextField
- DatePicker
- TimePicker
- Tabs
- List
- Avatar
- Divider
- BottomSheet
- SideSheet
- Snackbar
- FAB
- NavigationBar
- Progress

## کاربرد

این صفحه از این به بعد benchmark داخلی پروژه است.

هر بار که توکن‌ها، spacing، typography یا کامپوننت‌ها تغییر کنند، این صفحه باید دوباره بررسی شود.
