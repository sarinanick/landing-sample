# Private Security Notes

## نکته اصلی

این پروژه خصوصی است و نباید به صورت public منتشر شود مگر اینکه خود کاربر بعداً تصمیم بگیرد.

## موارد حساس

- لینک فایل Figma
- نام پروژه‌های شخصی
- design tokens اختصاصی
- رنگ برند
- فونت‌های لایسنس‌دار
- package registry خصوصی
- توکن‌های GitHub یا npm

## قوانین

- هیچ token یا secret داخل repo ذخیره نشود.
- اگر package خصوصی شد، registry باید private باشد.
- فونت‌ها داخل خروجی عمومی قرار نگیرند مگر مجوز داشته باشند.
- فایل‌های Figma فقط به عنوان reference استفاده شوند.
