# Storybook Guide

## هدف

Storybook مرکز مستندسازی و تست کامپوننت‌ها خواهد بود.

## ساختار Storyها

برای هر کامپوننت:

```txt
Component.stories.tsx
```

حداقل storyها:

- Default
- Variants
- States
- Disabled
- RTL
- Long text
- With icon
- Error اگر وجود دارد

## وضعیت فعلی

در v0.31 فقط scaffold اضافه شده است.

Storyهای نمونه:

- Button
- Icon
