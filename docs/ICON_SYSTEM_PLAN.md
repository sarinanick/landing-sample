# Icon System Plan

## وضعیت فعلی

در نسخه‌های قبلی برای سرعت توسعه از emoji/text icon استفاده شد.

## مشکل

- اندازه‌ها یکسان نیستند.
- وزن بصری متفاوت است.
- Pixel-perfect با Figma سخت می‌شود.

## راه‌حل

در v0.31 کامپوننت `Icon` به عنوان wrapper اضافه شد.

در آینده:

- SVG رسمی اضافه می‌شود.
- Material Symbols یا sprite داخلی استفاده می‌شود.
- تمام emojiها به Icon رسمی تبدیل می‌شوند.
